# Tetris
this is a TETRIS game writen in C with raylib.
an example project of my raylib&raygui bootstrap cmakelist as well.

## How to play?
S to speed the falling up, 
A to translate the tetromino left,
D to translate the tetromino right,
Z to CCW rotate the tetromino,
C to CW rotate the tetromino.

thanks for your playing.
